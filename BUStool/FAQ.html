What browser to use?

Google Chrome

Outlook for some reason interacts better with Chrome than Firefox. Therefore copying a table with styles will be only available when using Chrome.


How to use the tool?

Wisely

The tool was made to help you with Business Unit Setup, however it doesn't have data validation fields. Make sure you take following into consideration:
1. Make sure to check previous SPF records for E-commerce customers and adjust records to needs of a client,
2. Always check if all the data is correct with DNS verification tool
3. In DNS verifcation tool you will need to manually delete "/" sign from the DKIM key to make it work in "verify DNS" panel in ExpertSender
4. There is only version for four IPs in DNS instructions, make sure to check how many IPs are being assigned and adjust accordingly
